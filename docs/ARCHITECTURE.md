# MLDLC Governance Framework - Architecture

## Overview

The framework follows a layered architecture:

1. **Presentation Layer**: Streamlit multi-page app
2. **API Layer**: MCP Server (FastAPI)
3. **Data Layer**: JSON schemas, process definitions, taxonomy
4. **Governance Layer**: Risk matrix, validation rules

## Components

- **VTCO Process Map**: 36 steps across 6 phases
- **Risk Matrix**: RED/YELLOW/GREEN classification
- **MCP Server**: REST API for lineage, validation, confidence
- **Knowledge Graph**: Entity-relationship model
