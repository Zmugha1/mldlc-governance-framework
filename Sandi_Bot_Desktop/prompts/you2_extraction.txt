You are extracting data from TES You 2.0 profile
and TUMAY intake forms. The Entrepreneur's Source
uses these forms with prospective franchise clients.
The text may contain one or both documents combined.

ONE YEAR VISION:
Find the question that starts with:
"If we looked at your life a year from today,
what has to have happened during that period"
The answer follows immediately.
It may be one paragraph or two separate lines
starting with "Professionally" and "Personally"
Concatenate all answer text into one string.

Example format: [Client's actual vision from their document]
May be one paragraph or split into Professionally/Personally lines.

TOP 3 DANGERS — CRITICAL RULE:
Find the section titled "Top 3 Dangers & Goals"
Each entry has TWO lines:
Line 1: "Danger: [danger text]"
Line 2: "Goal: [goal text]"
Extract both lines as a paired object.
Do NOT use the Yes/No checkbox list above.
Clients frequently write freeform here even
when all checkboxes are No.

Example format: Danger: [client text]  Goal: [client text]

TOP 3 STRENGTHS — CRITICAL RULE:
Find section titled "Top 3 Strengths & Goals"
Same format: "Strength:" line + "Goal:" line
Extract as paired objects.

TOP 3 OPPORTUNITIES — CRITICAL RULE:
Find section titled "Top 3 Opportunities & Goals"
Same format: "Opportunity:" line + "Goal:" line
Extract as paired objects.

SPOUSE NAME:
Find "Question 2 - Who Else?"
"If you have a spouse or significant other,
what is their name?"
Extract the name that follows.

SPOUSE ROLE:
Find "Will your spouse...have a role
(ownership and/or employment)..."
Answer is Yes, No, or Unsure.
Map: Yes → "owner", No → "none", Unsure → "unsure"

SPOUSE ON CALLS:
Find "Will they be involved in future calls
or meetings?"
Answer is Yes or No.
Map to "yes" or "no"

SPOUSE MINDSET:
Find "Their mindset about business ownership
as an option:"
Extract the full paragraph that follows verbatim.
Example: [Client's spouse mindset verbatim from their document]

CREDIT SCORE:
Find "Estimated Credit Score:"
Extract the number only. Example: [Client's actual credit score]

NET WORTH:
Find "Estimated Net Worth:"
Extract exactly as written. Example: [Client's actual net worth range]
Do not reformat or add symbols.

LAUNCH TIMELINE:
Find "If you were to choose self-sufficiency,
what would be your ideal launch date?"
Or find "Selected: 0 - 6 months" pattern.
Extract the timeline text.

TIME COMMITMENT:
Find "Selected:" near time commitment options.
May include multiple selections separated by @
Example: "Full-Time - Owner Operated@Semi-Absentee
at Launch@Semi-Absentee after 2 years"
Convert @ separators to semicolons.

REASONS FOR CHANGE:
Find "Question 3 - Reasons for Exploring
Career Transition"
Extract ONLY items where value is "Yes"
Common items:
"I'm tired of the corporate world"
"I want more independence"
"Improving my lifestyle is important"
"I'd like to increase my income"
"I want to increase my wealth & equity"
"I'd like to own my own business"
"I want to make a difference in my community"
"I desire more flexibility"

SKILLS — CRITICAL (multiple label variations):
The You 2.0 document uses different labels for skills across clients.
Extract ALL professional capabilities into the JSON "skills" array.

1) Checkbox list (when present):
Find "Question 6 - Skills and Abilities"
Extract ONLY items where value is "Yes"

2) Freeform sections — scan the full document for ANY of these headings
or lead-in labels (case-insensitive; match partial lines):
- "Skills"
- "Transferable Skills"
- "Key Skills"
- "Abilities"
- "Core Competencies"
- "Professional Skills"
- "Skill Set"
- "What I Bring"
- "My Strengths" (when listed as bullet items of capabilities — not the
  paired "Strength:" / "Goal:" rows in "Top 3 Strengths & Goals"; those belong in strengths[])
- Any section that lists capabilities the client has developed professionally

Split items on commas, semicolons, or line breaks. Strip bullets (•, -, *).
Trim whitespace. One skill per array element. De-duplicate near-duplicates.

If no dedicated skills section exists,
extract implied skills from the Top 3
Strengths Goals text. Look for specific
capabilities mentioned in goal statements
like "Utilize strengths of relationship
building, trust, hard work and compassion"
and extract the skill phrases as an array.

FEW-SHOT EXAMPLES — SKILLS:

Example 1 — labeled "Transferable Skills":
Input: "Transferable Skills: Strategic thinking,
project management, structured execution,
sound decision-making"
Output skills: ["Strategic thinking",
"Project management", "Structured execution",
"Sound decision-making"]

Example 2 — labeled "Key Skills":
Input: "Key Skills: Relationship building,
leadership development, needs-based solutions,
mentoring and training"
Output skills: ["Relationship building",
"Leadership development",
"Needs-based solutions",
"Mentoring and training"]

Example 3 — unlabeled bullet list after
strengths section:
Input: "• Analytical thinking
- Data interpretation
- Cross-functional collaboration
- Process improvement"
Output skills: ["Analytical thinking",
"Data interpretation",
"Cross-functional collaboration",
"Process improvement"]

Example 4 — skills implied in strengths goals:
Input: "Strength: I have strong transferable
skills. Goal: Utilize strengths of
relationship building, trust, hard work
and compassion in a new and exciting way."
Output skills: ["Relationship building",
"Trust", "Hard work", "Compassion",
"Transferable skills"]

Example 5 — skills implied across multiple
strength goal statements:
Input: "Goal: Work with passion and joy
for the next 10 years.
Goal: Find a passion, work hard and smart
and succeed."
Output skills: ["Passion-driven work",
"Strategic thinking", "Persistence"]

If no skills can be identified anywhere
in the document return an empty array []
Do NOT return null.
Do NOT return a string.
Always return a valid JSON array.

AREAS OF INTEREST:
Find "Question 11 - Areas of Interest"
Extract ONLY items where value is "Yes"

LOCATION PREFERENCE:
Find "Question 4 - Ideal Location / Environment"
Rankings are 1 (most favorite) to 6 (least)
Extract the option with rank number 1.
If no rank 1 exists use the lowest rank number.
If tied concatenate both with " / "

PRIOR BUSINESS EXPERIENCE:
Find "Question 8 - Self-Sufficiency as an Option"
First freeform answer field.
Extract full text. Empty string if blank.

SELF SUFFICIENCY EXCITEMENT:
Find "As you explore Your Career 2.0,
what excites you about exploring
self-sufficiency as an option?"
Extract full text. Empty string if blank.

ADDITIONAL STAKEHOLDERS:
Find "Is there anyone else that would potentially
have a role in your possible new business venture?"
If Yes extract names and relationships.
Example: [Client's additional stakeholders from their document]
Empty array if No or none listed.

Return ONLY valid JSON. No explanation.
No markdown. Start with { end with }
